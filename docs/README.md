## 🎯 ما هي منصة تكامل؟

منصة تكامل هي **منصة أدوات ذكاء اصطناعي شاملة** باللغة العربية، تقدم 40+ أداة AI في مكان واحد:

- 💬 **دردشة ذكية** - شات بوت متقدم يدعم العربية
- ✍️ **أدوات محتوى** - كتابة مقالات، إعلانات، سوشيال ميديا، SEO
- 🎨 **أدوات صور** - توليد صور، إزالة خلفيات، تحسين جودة
- 🎬 **أدوات فيديو** - توليد فيديو من نص/صورة، lip sync
- 🎵 **أدوات صوت** - تحويل نص لصوت، إزالة ضوضاء
- 🏢 **أدوات أعمال** - تحليل منافسين، بحث سوق

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام | لماذا هذا الاختيار |
|---------|----------|-------------------|
| **Next.js 14+** | Frontend + API | App Router، SSR، أداء عالي |
| **TypeScript** | اللغة البرمجية | Type safety، أقل أخطاء |
| **Tailwind CSS** | التصميم | مرونة عالية، RTL support |
| **Framer Motion** | الحركات والأنيميشن | أنيميشنات سلسة |
| **PostgreSQL** | قاعدة البيانات | موثوقة، قوية |
| **Prisma** | ORM | Type-safe database queries |
| **NextAuth.js** | المصادقة | تسجيل دخول آمن |
| **Stripe** | الدفع | اشتراكات + credits |
| **AWS S3** | تخزين الملفات | صور، فيديو، صوت |
| **Redis** | التخزين المؤقت | سرعة الاستجابة |

## 🚦 القواعد الذهبية للمشروع

> ⚠️ **لازم تلتزم بالقواعد دي دايماً:**

1. **لا ديمو ولا Mock** - كل حاجة شغالة بالكامل
2. **لا Placeholder** - كل عنصر له محتوى حقيقي
3. **عربي أولاً** - الواجهة عربية RTL مع دعم الإنجليزي
4. **Premium Design** - تصميم مبهر مش basic
5. **Mobile First** - متجاوب مع كل الشاشات
6. **Credit System** - كل أداة تخصم credits
7. **Error Handling** - معالجة أخطاء كاملة
8. **Loading States** - حالات التحميل لكل عملية
9. **اقرأ الـ docs** - قبل ما تعمل أي حاجة اقرأ الملف المناسب
10. **حدّث الـ CHECKLIST** - بعد ما تخلص أي خطوة

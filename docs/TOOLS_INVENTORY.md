## 💬 الفئة 1: الدردشة والمساعد الذكي

### أداة 1: الشات الذكي (AI Chat) ⬜
- **المسار**: `/dashboard/chat`
- **الوصف**: شات بوت ذكي ثنائي اللغة يفهم العربية والإنجليزية
- **المدخلات**: نص المستخدم (رسالة)
- **المخرجات**: رد ذكي مع دعم Markdown و Code highlighting
- **الـ API**: OpenAI GPT-4o / Claude API
- **الكريديت**: 1 credit للرسالة العادية، 2 للرسالة الطويلة
- **المميزات**:
  - Streaming في الوقت الحقيقي
  - حفظ تاريخ المحادثات
  - إنشاء محادثات متعددة
  - تحميل الملفات والصور
  - نسخ الردود
  - تقييم الردود (إعجاب/عدم إعجاب)
- **ملفات التطوير**:
  - `src/app/(dashboard)/chat/page.tsx`
  - `src/components/chat/ChatInput.tsx`
  - `src/components/chat/ChatMessage.tsx`
  - `src/components/chat/MessageList.tsx`
  - `src/app/api/chat/route.ts`

### أداة 4: كاتب المقالات الاحترافي (Article Writer Pro) ⬜
- **المسار**: `/dashboard/tools/content/article-writer`
- **الوصف**: كتابة مقالات SEO احترافية خطوة بخطوة
- **المدخلات**:
  - عنوان المقال أو الموضوع
  - الكلمات المفتاحية (اختياري)
  - طول المقال (500/1000/2000/3000 كلمة)
  - نبرة الكتابة (رسمي/غير رسمي/أكاديمي/تسويقي)
  - اللغة (عربي/إنجليزي)
- **المخرجات**: مقال كامل بتنسيق HTML/Markdown
- **الـ API**: OpenAI GPT-4o
- **الكريديت**: 5-20 credits حسب الطول
- **Prompt Template**:
```
أنت كاتب محتوى محترف متخصص في SEO. اكتب مقال شامل عن:
الموضوع: {topic}
الكلمات المفتاحية: {keywords}
الطول: {length} كلمة
النبرة: {tone}
اللغة: {language}

المقال يجب أن يتضمن:
- عنوان جذاب
- مقدمة تشد القارئ
- عناوين فرعية H2 و H3
- فقرات منظمة
- خاتمة مع دعوة للعمل
- استخدام الكلمات المفتاحية بشكل طبيعي
```
